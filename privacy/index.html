Privacy

Here you can find our privacy policy.

Last update: 14.07.2024 - 13:50

  1. Sites

    • On some of our websites we use cookies to store user data and improve the user experience.

  2. Baxi

    • Baxi works closely with the Cara system. Cara handles the identification of spam messages and users.
      The following data is used to provide these services:
      • Guild Name
      • Guild ID
      • User Name
      • User ID
      • Channel Name
      • Channel ID
      • Message
      • Message ID
      • Image link (if available)
    • This data is processed on all servers when a message is sent.
    • Images sent in channels that Baxi has access to are uploaded to our server and given an individual ID. This link is forwarded to our partner program "Cara" to support future systems such as image analysis for the chat filter.
      • You can disable this by running the command /privacy option:disable setting:image_processing on your server.
    • All messages are processed independently of the guild configuration and may be stored if the data is relevant.
    • Only relevant data that is needed to review or improve our services will be stored.
    • The new system "Cara" creates a detailed behavior profile based on your messages. You can find more detailed Privacy Policy information here.
    • All systems that process and/or collect data together with our systems run locally on PyroPixles's own servers (SkyFire Hosting), and are only accessible to authorized personnel.
    • IMPORTANT: We and our partner system Cara do not share, sell or publish any information that we process and/or collect.
    • To request deletion of your current user profile, you must join our support server and create a support ticket.
    • All messages sent in GlobalChat and their IDs as well as the ID of the author are stored on our servers.
      This has the following reasons:
      • Moderation: Our GlobalChat offers our team and users the option to delete messages. The individual IDs are required for this. The IDs are removed from the database after the desired message has been deleted.
    • All files sent in the global chat are automatically uploaded to our servers and given their own ID. These files can then be accessed via a link.
      • Feature: This is done to display pictures in the global chat so that users can easily send pictures.

  3. Hosting

    • All customer servers hosted on our hosting "SkyFire" are always accessible to authorized personnel. This is for the following reasons:
      • Moderation: Our team performs random checks to ensure that nothing is hosted on our servers that violates the law or guidelines.
      • Support: Our team has access to the customer servers to provide assistance as quickly as possible if problems arise.

  4. Contact

    • If you have any questions or concerns, our support is always available via email (currently unavailable) or on our Discord support server!.
